

全局安装typescript包

```bash
npm install typescript -g

# npm install typescript -g --force
```



生成配置文件

```bash
tsc --init
```



用tsc编译ts文件

输入`tsc`会把当前目录的所有`.ts`文件都编译成`.js`文件

输入`tsc test.ts`会把对应路径的`.ts`文件编译成`.js`文件



> 想使用code runner即时运行ts,需安装全局包`npm install ts-node -g`
